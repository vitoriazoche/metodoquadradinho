# método quadradinho
    é um método de estudo adaptado pela influencer Luana Carolina em que o estudante lista pequenos temas dentro de um assunto para serem estudados ou revisados, e ao invés de marcar com um X, ele pinta caixinhas para indicar o seu progresso. Como mostrado na imagem abaixo:


![](https://pbs.twimg.com/media/ExMXgRJVgAQm7Gg?format=jpg&name=medium)

Como forma de treinar meus conhecimentos em desenvolvimento web e criatividade, resolvi tornar este método digital como forma de **desafiar**. Você também pode utilizá-lo se quiser, para isso siga as instruções abaixo:

1. Dê Fork nesse repositório
2. Vá até o `seuusuario/metodoquadradinho` e clique em **code** 
3. Clique em copiar HTTP
4. Vá até o git e execute a seguinte linha
`git clone https://github.com/seuusuario/metodoquadradinho.git`

Caso queira só acompanhar as novas atualizações desse repositório clique em **Watch** e ficará por dentro das atualizações!!!
